Deploy your Node.JS applications to Nodester for free, or deploy them to your own private cloud instance of Nodester running in your own datacenter, or on Amazon EC2, Rackspace, or GoGrid!


Nodester is an open source Node.JS platform-as-a-service, written in Node.JS with a RESTful API and designed to run in any cloud without a VPS! It is the first of its kind and 100% open source under the GNU Affero license on GitHub.
